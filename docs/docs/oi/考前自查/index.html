⭐ 考前自查

考前 Checklist。

下载试题、提交程序​

下载试题：

  1. 连接 FTP 服务器，地址形如：ftp://192.168.xxx.xxx。
  2. 选择所有题目，右键复制，导航到桌面，右键粘贴。
  3. 建议直接在每道题目的子文件夹中创建 .cpp 文件。

提交程序（阅读桌面上的 README.pdf）：

  1. 每道题目建立一个子文件夹。
  2. 子文件夹应当仅包含一个 .cpp 文件。

提交前检查​

  1. freopen() 写了吗？文件名写对了吗（测完样例文件名改回来没有）？逐字符检查。 正确示例：
freopen("bracket.in", "r", stdin);
freopen("bracket.out", "w", stdout);

  1. .cpp 文件名对了吗？题目文件夹名字对了吗？逐字符检查。

  2. 检查输出语句，调试代码真的删干净了吗？

  3. 如果题目有多组测试数据，真的把所有状态清空了吗？

编译、调试​

可以参考下面的编译参数，或者复制试题给的参数（最好加上 -Wall）。

g++ -O2 -Wall -std=c++14 -static

运行程序并测试用时：

time ./a.out

数据类型​

  1. int（%d）最大值：231−1≈109.332^{31}-1\approx10^{9.33}231−1≈109.33，数字不大于 10910^9109 可用。
  2. long long（%lld）最大值：263−1≈1018.962^{63}-1\approx10^{18.96}263−1≈1018.96。
  3. unsigned long long（%llu）最大值：264−1≈1019.272^{64}-1\approx10^{19.27}264−1≈1019.27。用 ull 必检查：过程是否会出现负数？

STL​

结构体排序​

要让自定义的结构体能被 STL 排序，需要重载小于号 <：

sort 升序，priority_queue 大根
struct MyStruct {
    int i;
    bool operator<(const MyStruct& another) {
        return i < another.i;
    }
}

sort()​

sort() 默认是升序。

sort(begin, end); // 升序
sort(begin, end, greater<int>()); // 降序

sort(a, a + n); // 用 a[0...n-1]
sort(a + 1, a + n + 1); // 用 a[1...n]
sort(v.begin(), v.end());

begin, end 表示排序的开始和结束范围，即 [begin, end)。

priority_queue​

priority_queue 默认是大根堆。

priority_queue<int> q1; // 大根
priority_queue<int, vector<int>, greater<int>()> q2; // 小根

时间复杂度1​

评测机在 1s 内约可以完成 10810^8108 次操作。基于此估计可以估计各时间复杂度能通过的数据规模。

详细估计
  • n<10n<10n<10：O(n!)O(n!)O(n!) - 排列问题
  • n<15n<15n<15：O(2n)O(2^n)O(2n) - 组合问题
  • n<50n<50n<50：O(n4)O(n^4)O(n4) - DP
  • n<200n<200n<200：O(n3)O(n^3)O(n3) - DP，全对最短路径
  • n<103n<10^3n<103：O(n2)O(n^2)O(n2) - DP，全对问题，稠密图
  • n<106n<10^6n<106：O(nlog⁡n)O(n \log n)O(nlogn) - 排序贪心，堆，分治
  • n<106n<10^6n<106：O(n)O(n)O(n) - DP，图遍历 / 拓扑排序 (V+E)(V+E)(V+E)，树遍历
  • n<109n<10^9n<109：O(n)O(\sqrt{n})O(n​) - 质数判定，平方和问题
  • n<109n<10^9n<109：O(log⁡n)O(\log n)O(logn) - 二分答案
  • n<109n<10^9n<109：O(1)O(1)O(1) - 数学公式计算

内存估计​

在程序开头和 main() 前各定义一个 bool，两者内存地址相减就是（全局）使用的内存大小。例如：

#include <bits/stdc++.h>
bool aaaa;
int s[100005];
bool bbbb;
int main() {
    cout << float(&bbbb - &aaaa) / 1024.0 / 1024.0 << "MB\n";
    return 0;
}
