题意简述​

给两个数字 a,ba,ba,b，输出 a+ba+ba+b。

范围：−109≤a,b≤109-10^9\le a,b\le10^9−109≤a,b≤109。

    cin >> a >> b;
    cout << a + b;
    return 0;
}
