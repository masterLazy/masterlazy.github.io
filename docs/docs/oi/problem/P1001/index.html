题意简述​

给两个数字 a,ba,ba,b，输出 a+ba+ba+b。

范围：−109≤a,b≤109-10^9\le a,b\le10^9−109≤a,b≤109。

思路​

这么简单的题，还有啥思路。。。

编码​

用时 41 ms 内存 1.04 MB
/*
* P1001 A+B Problem
*/
#include <bits/stdc++.h>
using namespace std;
int main() {
    int a, b;
    cin >> a >> b;
    cout << a + b;
    return 0;
}
